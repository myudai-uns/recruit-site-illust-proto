# アイシン高丘東北 採用HP FV案 — 線画水彩イラスト版

新卒向け採用HPのファーストビュー（FV）リデザイン案。プロデザイナーのFB「FVが弱い／3カラムは良い」を受け、3カラム縦スクロールパズル構造は維持しつつ、タイル内容を実写 → 線画＋水彩タッチのイラストに差し替えたプロトタイプ。

- **元素材**: `~/recruit-site/`（書き換えず別管理）
- **イラスト生成**: ChatGPT × Playwright（`generate_images.js` 参照）
- **イラスト仕様**: 1254×1254px PNG、12枚、線画＋水彩タッチ
